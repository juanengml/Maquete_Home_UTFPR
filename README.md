# Maquete_Home_UTFPR
Controle e Prototipação da maquete da casa

## Grupo de Pesquisa de IoT da UTFPR-TD
### Participantes: 
- Juan Manoel (Eng Computação) 
- Debora Rodrigues (Eng Eletronica) 
## Problematica: 
* Altos indices de uso de energia de forma irregular
* Lampadas acessas em horarios que não deveria


## Objetivol
* Monitorar o Ambiente com sensores de luminosidade e presença
* Controlar Ambientes de forma remota EX:(Lampadas)
* Controlar Garagem 
* Criar dashbaord para sincronização do dispositivo(Arduino Mega+ EthernetShield)


## Prototipo 
* Construir uma dashboard para ler os dados 
* Prototipar um IoT
-  Garagem(Servo motor 9g)
-  Lampandas (leds pela maquete)
- sensor de luz (controle automatico com LDR)
- sensor de movimento(controle automatico com PIR
